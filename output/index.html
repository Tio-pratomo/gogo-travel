Tentang GoGoTravel

GoGoTravel adalah perusahaan yang melayani pemesanan hotel, tiket pesawat dan tiket kereta api melalui situs web serta aplikasi gratis di Android dan IOS.
GoGoTravel akan mengelola kebutuhan liburan dan perjalanan dengan menjadi Fun Travel Partenering bagi para travelers Indonesia.
